Sudoku Solver
A program that will take a given sudoku board then output whether it is solvable or not, and
displaying the solution if it is solvable.

Run Requirements:
pip install pygame

To use sudoku.py, fill out the board with the desired sudoku board and start. For the GUI version,
input the desired sudoku board onto example board, run the program and click space to watch the 
algorithm solve the problem.

Note that this project was created in July 2022 but never uploaded until now.
